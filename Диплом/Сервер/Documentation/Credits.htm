QR-Code Studio is based in part on the work of the projects listed below - TEC-IT says "Thank You" to the respective developers!

In each case you agree to be charged with a reasonable amount compensating our handling & distribution efforts.

  • Data Compression Library - zlib
    For details please refer to http://www.zlib.net

  • Character-Set Conversion Library - GNU Iconv
    For details please refer to http://www.gnu.org/software/libiconv/
    License: LGPL 2. See file iconv.LICENSE in the Bin directory of the QR-Code Studio installation. This file is identical to the file COPYING.LIB which is part of the library source code.
  • Secure Sockets Layer and Transport Layer Security - OpenSSL
    For details please refer to http://www.openssl.org/
    License: See file openssl.LICENSE in the Bin directory of the QR-Code Studio installation.
  • Social Media Icons
    The great icons used in the info-box were designed by chethstudios - see http://chethstudios.blogspot.com/.
