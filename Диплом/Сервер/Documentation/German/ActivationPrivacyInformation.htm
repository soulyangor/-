Online Aktivierung

Was ist Online-Aktivierung?

Das Aktivieren von TEC-IT Software bedeutet im einzelnen, dass die Software via Internet mit dem Aktivierungsserver von TEC-IT Kontakt aufnimmt, die unten angeführten Daten online an diesen Aktivierungsserver überträgt und daraufhin einen Lizenzschlüssel vom Aktivierungsserver empfängt. Dieser Lizenzschlüssel schaltet das Produkt dann dauerhaft frei.
Der Aktivierungsserver dient ausschließlich zum automatisierten Verteilen der Lizenzschlüssel an Kunden und zur Verwaltung der einzelnen Aktivierungen. Der Aktivierungsserver (und somit eine Internet-Verbindung) wird im Normalfall nur einmal zur Produktfreischaltung benötigt.

Welche Information werden an TEC-IT gesendet?

Folgende Daten werden bei der Aktivierung an TEC-IT gesendet und wenn nötig dauerhaft bei TEC-IT gespeichert:

  • Aktivierungsschlüssel
    Dieser Aktivierungsschlüssel wurde Ihnen von TEC-IT (oder einem TEC-IT Partner) nach Bestellung der Software zugesandt. In diesem Schlüssel sind keine Daten (welcher Art auch immer) kodiert.
  • Firmenname, Land, Postleitzahl und Stadt
    Diese Daten sind erforderlich um Ihnen eine Lizenz zuordnen zu können.
  • EMail-Adresse
    Eine Kopie der Lizenzdaten wird an diese Email-Adresse gesendet.
  • Produkt-IDs
    Die Produkt-IDs kennzeichnen das Produkt, dass aktiviert werden soll.
  • System-ID
    Die System ID kennzeichnet den Computer auf dem das Produkt aktiviert wird. TEC-IT verwendet als System-ID entweder die Seriennummer des Datenträgers (Festplatte), die Seriennummer des Rechners oder den Host-Namen (der Name des Computers im Netzwerk).
  • Produkttexte
    Lokalisierte Bezeichnungen für Produkt und verfügbare Lizenzvarianten.
  • IP Adresse, Datum und Uhrzeit
    Die IP Adresse Ihres Computers (oder des Internet-Gateways) sowie Datum und Uhrzeit werden aus Gründen der Nachvollziehbarkeit gespeichert.

Gibt TEC-IT Informationen weiter?

Generell werden Daten wie Name, Email-Adresse, Stadt, ... vertraulich behandelt. Eine Weitergabe an Dritte erfolgt nur unter folgenden Voraussetzungen:

  • Wenn dies gesetzlich vorgeschrieben ist
  • Wenn dies für die Abwicklung Ihrer Bestellung notwendig ist
    Z.B. die Weiterleitung von Daten zur Zahlungsabwicklung.
  • Oder wenn Sie der Weiterleitung Ihrer persönlichen Daten vorher zustimmen.
    Etwa wenn wir Sie gerne als Referenzkunde erwähnen wollen. Sie werden in solchen Fällen von TEC-IT um Erlaubnis gebeten.

Manuelle Aktivierung (ohne Internet)

Kann ich auch eine alternative Lösung zur Aktivierung verwenden?

Ja. TEC-IT bietet auch eine manuelle Methode zur Produktaktivierung an, kontaktieren Sie uns einfach! Diese Methode benötigt keine Internetverbindung. Die System-ID ist zur Berechnung der Lizenz fallweise nötig (produktabhängig).
